    • Jsopoly

    Jsopoly is a Javascript platform for building a community driven site to share news, experiences and information in a digg's style

    Project maintained by mockguiver Hosted on GitHub Pages — Theme by mattgraham

    Welcome to Jsopoly

    Jsopoly is a early v.0.0.2 development with focus on provide a tool to share and rate javascript libraries, posts, enhancements, gists, etc. It's on early development process so expect big changes and bug fixes.

    Main technologies include:

    * node.js / express backend, with MongoDB / Mongoose database
* ONLY web sockets with socket.io communication
* Angular.js / Bootstrap on client

    It implements crypto messages and session storage via LocalStorage (with no libraries). Don't expect it to work in old browsers.

    The platform

    The platform is hosted on http://jsopoly.eu01.aws.af.cm and it could be outdated.
